# gav.fi tools

Personal collection of visual experiments and standalone tools.

Deployed via GitHub Pages: **https://gavinhaggis.github.io/tools/**

Repo: https://github.com/gavinhaggis/tools

**Homepage:** [index.html](index.html) — tool gallery with CSS preview cards.

## Tools

All tools live in `tools/`. Add a new tool by dropping an HTML file there and adding a card to `index.html`.

| File | Description |
|------|-------------|
| [tools/credits.html](tools/credits.html) | Film-style credits PNG exporter |
| [tools/dvd-bounce.html](tools/dvd-bounce.html) | DVD screensaver with control panel |
| [tools/mav-generator.html](tools/mav-generator.html) | Mav pixel mascot colour customiser |
| [tools/qr-pixel-generator.html](tools/qr-pixel-generator.html) | Pixel-art QR code generator |
| [tools/social-media.html](tools/social-media.html) | AV comment card / social asset generator |
| [tools/team-timer.html](tools/team-timer.html) | Countdown timer with team name display |
| [tools/zoetrope.html](tools/zoetrope.html) | Zoetrope animation strip creator |
| [tools/group-generator.html](tools/group-generator.html) | Smart group-splitting tool (Aalto EE) |
